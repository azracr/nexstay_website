# NexStay â€“ Hotel Finder & Booking System

NexStay is a hotel search, comparison, and booking web application developed using React and Vite.

The application allows users to:
- Search hotels by city, dates, and number of guests
- Filter and sort search results
- View hotel details (photos, amenities, reviews)
- Compare hotels side by side
- Complete a booking with a simulated payment
- View bookings stored locally

This project was developed as part of the Web Programming course final project.
