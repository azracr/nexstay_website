NexStay is a hotel search, comparison, and booking web application developed using **React** and **Vite**.  
The project allows users to search hotels by city, compare up to three hotels, and make bookings that are stored locally.

This project was developed as part of a course assignment.

---

## Installation & Run

1. Clone the repository:
   git clone https://github.com/azracr/nexstay_website.git

2. Install dependencies:
   npm install

3. Run the project:
   npm run dev

The application will run on http://localhost:5173


##  Technologies Used

- React
- Vite
- React Router
- JavaScript (ES6+)
- CSS
